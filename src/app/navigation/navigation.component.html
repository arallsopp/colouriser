About

This is a quick tool built out on a retina iMac. You may find your machine renders colours differently.

All colours vary between screens, so ignore the presets if they're nonsense.

Mobiles seem not to want to render any changes until you rotate the device. Sorry.

How to use

Set the sliders wherever gets you the effect you want.

Use the buttons to copy from body to roof or vice-versa.

Print screen your favourites.
